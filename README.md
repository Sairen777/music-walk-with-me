# Music Walk With Me

Pick a year and hear what American teenagers were listening to, with films,
games, gadgets, and period-accurate device skins wrapped around the music.
Coverage is **USA 1990–2007**: choose a year, drop into a capsule, and a
30-second preview plays over a wall of real album covers inside the device of
the time.

## Run it

```bash
npm install
npm run dev      # http://localhost:5173
```
